# persona-website

Portfolio website for **Prateek Kumar**.

## Stack
- React (Vite)
- React Router (multi-page navigation via `navigate()`)
- MUI
- SCSS

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Content
Profile content lives in:
- `src/content/profile.json`
- `src/content/site.json`
- `src/content/resume.txt`
